###
Notification-receiver

Environment Variables for local
```
NODE_ENV=local
REMOTE_FUNCTION=flux-event
AWS_SDK_LOAD_CONFIG="true"
```

Environment Variables for non-local
```
NODE_ENV=staging|production
REMOTE_FUNCTION=flux-event
AWS_SDK_LOAD_CONFIG="true"
```
